✅ Day 2 – HTML Table & Tasks (Code 1)
<!doctype html>
<html lang="en">
<head>
  <!-- charset & viewport already explained in Day 1 -->
  <title>Basic Table</title>
</head>

<body>

  <h1>This is basic table</h1>

  <ujwal>this is Ujwal</ujwal>
  <!-- 
  This is a custom / unknown tag.
  HTML allows it, but it has no meaning.
  Used only for learning, not real projects.
  -->

  <table border="1">
    <!-- 
    <table> creates a table structure.
    border="1" is an attribute used to show table borders.
    -->

    <thead>
      <!-- 
      <thead> groups table heading rows.
      Helps browser understand table structure.
      -->

      <tr>
        <!-- <tr> defines one row in a table -->

        <th>Name</th>
        <th>Roll</th>
        <th>City</th>
        <!-- 
        <th> is table heading cell.
        It is bold and centered by default.
        -->
      </tr>
    </thead>

    <tbody>
      <!-- 
      <tbody> contains actual table data.
      -->

      <tr>
        <td>Rohan</td>
        <td>111</td>
        <td>Mohali</td>
        <!-- 
        <td> is table data cell (normal text).
        -->
      </tr>

      <tr>
        <td>Rohit</td>
        <td>222</td>
        <td>Pune</td>
      </tr>

    </tbody>
  </table>

</body>
</html>

✅ Day 2 – Image + Concepts (Code 2)
<!doctype html>
<html lang="en">
<head>
  <title>Second Class</title>
</head>

<body>

  <h1>this is heading</h1>
  <p>this is normal text</p>

  <ul>
    <!-- 
    Unordered list usage revised here.
    -->
    <li>guava</li>
    <li>mango</li>
    <li>orange</li>
  </ul>

  <img src="./images/favicon.png" alt="site icon" />
  <!-- 
  <img> is a self-closing tag.
  src = image location
  alt = alternative text if image fails to load
  -->

  <p>this is your second class</p>

</body>
</html>



1️⃣ HTML TAG
A tag tells the browser what to do with content.
Example: <p>, <table>, <img>

2️⃣ HTML ELEMENT
Everything from opening tag to closing tag.
Example: <p>Hello</p>

3️⃣ HTML ATTRIBUTE
Extra information written inside the opening tag.
Example: src, alt, border, href
